Focus Styling

Spark Components and anchor tag elements by default use an accessible focus indicator style. Use the gux-focus-ring-focused and gux-focus-ring-focused-within classes to add Spark accessible focus indicator styles to elements that are not part of the Spark Component Library.

gux-focus-ring-focused

Add the `gux-focus-ring-focused` class to a focusable element to add the Spark focus ring

gux-focus-ring-focused-within

In some cases, the focus indicator might need to be shown on a parent of a focusable element. Use the `gux-focus-ring-focused-within` class on the element that you want to show the focus indicator around.

gux-focus-ring-focused-visible

Add the `gux-focus-ring-focused-visible` class to a focusable element to add the Spark focus ring
