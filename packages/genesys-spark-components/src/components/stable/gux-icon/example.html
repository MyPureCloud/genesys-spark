Shadow DOM

The `gux-icon` is expected to work even when used in the shadow dom (open)

Accessibility

Set the decorative attrribute if the icon does not convey information about the page and is only for decoration

Set the screenreader-text attribute if the icon conveys information to the user that would not otherwise be available

If neither decorative or screenreader-text are set an error will be logged in the console

Styling

You can use the color, height and width css properties to style your icon

Sizing

inherit (height and width set on element)

small (16x16)

medium (24x24)

large (32x32)

Caching

Duplicated icons should not require additional network calls
