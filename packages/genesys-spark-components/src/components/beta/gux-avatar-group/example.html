gux-avatar-group-beta

Summary

A group that can contain gux-avatar-group-item-beta tags. A tooltip will be displayed for each group item on focus or hover.

Group Focus Example

Add to Group

Slot a gux-avatar-group-add-item-beta tag at the end to display add to group option

Properties

quantity

The quantity property indicates the number of group items that will be displayed outside of the overflow menu.
