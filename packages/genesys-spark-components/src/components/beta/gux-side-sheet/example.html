Modal Side Sheet

You can show/hide the `gux-modal-side-sheet` via two mechanisms:

  1. Use the `showModal()` and `close()` methods on a `gux-modal-side-sheet` element
  2. Set the `open` property or attribute on `gux-modal-side-sheet`
