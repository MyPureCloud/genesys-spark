gux-tooltip-pointer-beta

Summary

Simple tooltip that is displayed when hovering over a linked node.

Default Behavior

By default, the tooltip will be displayed when hovering over the parent node. Default positioning is beneath the node. The tooltip is not restricted by the parent's overflow style. If the text is sufficiently long, it will wrap into multiple lines.
