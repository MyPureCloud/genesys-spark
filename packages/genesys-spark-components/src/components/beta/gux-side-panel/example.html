Modal Side Panel

You can show/hide the `gux-modal-side-panel` via two mechanisms:

  1. Use the `showModal()` and `close()` methods on a `gux-modal-side-panel` element
  2. Set the `open` property or attribute on `gux-modal-side-panel`
