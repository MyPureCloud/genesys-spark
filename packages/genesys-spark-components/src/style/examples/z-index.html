z-index values are based on the Spark elevation documentation
z-index value token components
0 gse-semantic-zIndex-base Default z-index value for basic elements
1 gse-semantic-zIndex-showFocus Apply to elements that need an elevated focus style control
1 gse-semantic-zIndex-sticky Used for sticky headers
100 gse-semantic-zIndex-popup Popups (used in dropdowns, date pickers, etc)
200 gse-semantic-zIndex-popover Popovers
200 gse-semantic-zIndex-tooltip Tooltips
300 gse-semantic-zIndex-panel Panels (reserved for future use)
400 gse-semantic-zIndex-navbar Top Navigation bars (reserved for future use)
400 gse-semantic-zIndex-sidebar Side Navigation bars (reserved for future use)
500 gse-semantic-zIndex-toast Toasts
600 gse-semantic-zIndex-modal Modals
